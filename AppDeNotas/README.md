Estas son las instrucciones básicas para usar la aplicación de Notas de texto. Puedes gestionar y organizar tus notas, así como personalizarlas según tus preferencias.

Se debe clonar el repositorio del backend y seguir las instrucciones.
https://github.com/zabdielrojas/ApiDeNotas

Clonar este repositorio.

Instalar las dependencias:
`npm i`

Iniciar el Backend
`npm run dev`

Iniciar el Frontend
`npm run dev`

**Para Usuarios Anónimos:**

1. **Login:** Inicia sesión utilizando tu dirección de correo electrónico y contraseña.
2. **Registro:** Si eres nuevo en la aplicación, puedes registrarte proporcionando tu dirección de correo electrónico y una contraseña.

**Para Usuarios Registrados:**

3. **Ver Listado de Notas:** Después de iniciar sesión, verás un listado de tus notas. En este listado, solo verás los títulos de las notas.
4. **Visualizar una Nota:** Haz clic en el título de una nota para ver su contenido completo.
5. **Crear una Nota:** Para crear una nueva nota, proporciona un título, un texto y elige una categoría única de entre las categorías fijas disponibles.
6. **Modificar Notas:** Puedes editar tus notas existentes, incluyendo el título, el texto y la categoría.
7. **Marcar una Nota como Pública:** Por defecto, todas las notas son privadas y solo las puedes ver tú. Sin embargo, si deseas que una nota sea pública y accesible por otros usuarios (registrados o no), puedes marcarla como pública. Ten en cuenta que las notas públicas solo se pueden acceder si se conoce la URL específica de la nota.
8. **Eliminar una Nota:** Si deseas eliminar una nota, tienes la opción de hacerlo.
9. **Asociar una Imagen:** También tienes la capacidad de asociar una única imagen a cada nota, lo que puede ayudarte a organizar y personalizar tus notas.
